# IoT Project 1 - Smart Environmental Monitor

## Objective
Measure temperature and humidity using an ESP32 and display the values on a 16x2 I2C LCD.

## Components
- ESP32 DevKit
- DHT22 Sensor (Simulation)
- 16x2 I2C LCD
- Wokwi Simulator

## Features
- Reads temperature and humidity.
- Displays values on the LCD.
- Prints values to the Serial Monitor.
- Updates every 2 seconds.

## Simulation
Developed and tested using Wokwi.
